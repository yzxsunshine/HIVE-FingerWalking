UIVA_Server

---------------------------Purpose----------------------- 
1. Connect to the VRPN servers of the devices;
2. Connect to UIVA_Client in Unity Indie;
3. Upon request from UIVA_Client, poll data from
   the VRPN servers and respond with latest sensor data.

-------------------------How to use-------------------------
1. Read the documentation of UIVA.
2. Read UIVA_Server.cfg.
3. Customize UIVA_Server.cfg for your own use.
4. Start UIVA_Server.exe after all device VPRN servers are started.
